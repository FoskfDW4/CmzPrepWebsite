CmzPrep Rev 2

基于 .NET 6 / WPF 构建的 Windows 系统重装工具，支持 WIM/ESD/ISO 映像部署，多语言界面，深色主题。

📦
WIM / ESD / ISO 支持
支持标准的 Windows 映像格式，自动从 ISO 中提取 WIM 文件，兼容 Win7-Win11 安装映像。
🖼
映像索引解析
自动读取 WIM 文件中的映像列表，支持版本名称中/英翻译，一键选择专业版。
💾
目标分区选择
自动列出非系统分区，显示文件系统类型、卷标、总容量和剩余空间。
🌐
三语言界面
内置 简体中文 / 繁體中文 / English 切换，即时生效，无需重启。
⚙
灵活配置
独立的选项窗口，支持版本翻译开关、自动选择、导出驱动、PE清理等配置。
📋
实时部署日志
三阶段进度指示器 + 实时日志窗口，部署过程全程可视，支持取消。

系统要求

  • 操作系统: Windows 7 SP1 及以上（64 位）
  • 运行时: .NET 6 Desktop Runtime（或使用单文件发布版）
  • 支持格式: .wim / .swm / .esd / .iso
  • 语言支持: 简体中文 / 繁體中文 / English
